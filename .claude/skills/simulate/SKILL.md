# Simulate Skill

## 触发方式

用户输入 `/simulate` 时触发此技能。

## 描述

运行仿真并自主修复所有问题，直到仿真完全正常为止。

## 执行流程

1. **运行仿真**
   - Windows: `cmd //c "Script\\sim.bat 120"`
   - macOS/Linux: `bash Script/sim.sh 120`

2. **读取日志**: `Logs/uav.log`

3. **分析日志**，结合源码，从以下维度评估：
   - **速度**：巡航速度是否达标，避障时速度损失是否合理
   - **稳定性**：姿态/速度是否震荡，控制输出是否平滑
   - **避障轨迹**：绕障路径是否流畅，有无不必要的大幅偏转或原地徘徊
   - **轨迹跟踪偏差**：所有 UAV 偏差是否在 3m 以内

4. **若发现问题**：
   - 定位根本原因（读取相关源码）
   - 实施修复
   - 编译：
     - Windows: `cmd //c "Script\\build.bat"`
     - macOS/Linux: `bash Script/build.sh`
   - 编译失败 → 读取错误输出 → 修复代码 → 重新编译
   - 编译成功 → 回到步骤 1

5. **每次迭代记录**：修改了什么、为什么修改

6. **仿真无错误且所有 UAV 轨迹偏差 ≤ 3m** → 输出最终总结并结束

## 约束

- 不要在迭代之间询问确认，保持自主运行
- 不要主动提交代码
